Política de Privacidad

Última actualización: 2026

Esta aplicación respeta la privacidad de los usuarios. La aplicación muestra anuncios y no recopila, almacena ni comparte datos personales del usuario, salvo la ubicación cuando es necesaria para mostrar la información de los widgets.

Uso de la cámara

La aplicación solicita acceso a la cámara del dispositivo únicamente para permitir el escaneo del código QR generado por la CNMC (Comisión Nacional de los Mercados y la Competencia).

Este código QR se utiliza exclusivamente para facilitar la búsqueda de ofertas de electricidad disponibles para el usuario.

La aplicación no guarda imágenes, vídeos ni ninguna información capturada por la cámara.

Datos personales

La aplicación no recopila datos personales del usuario, salvo la ubicación del dispositivo cuando resulta necesaria para mostrar la información de los widgets.

La ubicación se utiliza exclusivamente mientras la aplicación está en uso para ofrecer contenido relacionado con los widgets y no se conserva posteriormente.

No se almacena información personal ni en el dispositivo del usuario ni en servidores externos.

Servicios de terceros

La aplicación muestra anuncios mediante servicios de terceros. Fuera de ese uso publicitario, la aplicación no comparte información personal del usuario con terceros.

Cambios en esta política de privacidad

Esta política de privacidad puede actualizarse en el futuro para reflejar cambios en la aplicación o en requisitos legales. Cualquier actualización se publicará en esta misma página.

Contacto

Si tienes alguna pregunta sobre esta política de privacidad, puedes contactar con el desarrollador de la aplicación.
